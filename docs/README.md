# Documentation

This folder explains the design and security thinking behind the demo.

## Files
- `paper_draft.md` - the paper draft.
- `architecture.html` - the interactive architecture diagram.
- `threat_model.md` - attack scenarios and controls.
- `evaluation.md` - test results and measured behavior.

## Why It Exists
Use these files to understand the system, the threats it handles, and the evidence behind the design.
